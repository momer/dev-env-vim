LocalLeader Commands

Press ,, + key for language actions

,,r

Run file

,,t

Run tests

,,b

Build (Go/Rust)

Same keys work across Go, Ruby, JS/TS, Rust
Each language uses the same keybindings but different commands. ,,r runs the current file, ,,t runs tests, ,,b builds. For Rust, ,,c runs cargo check which is faster than build.

Markdown Preview

Opens a live-updating preview as you type
