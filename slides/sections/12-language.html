LocalLeader Commands

Press ,, + key for language actions

,,r

Run file

,,t

Run tests

,,b

Build (Go/Rust)

,,i

REPL (Elixir)

Same keys work across Go, Ruby, JS/TS, Rust, Elixir
Each language uses the same keybindings but different commands. ,,r runs the current file, ,,t runs tests, ,,b builds. For Rust, ,,c runs cargo check which is faster than build. For Elixir, ,,i opens iex in a terminal buffer. To exit terminal mode: Ctrl+\ then Ctrl+n.

Indentation Settings

Appendix

Go: 4 spaces (tabs) · Rust: 4 spaces

Ruby/JS/TS: 2 spaces · Elixir: 2 spaces

Set automatically when you open a file
Indentation is configured per-language in ftplugin files. Go uses tabs as per its conventions. Elixir uses 2 spaces per Elixir conventions. You don't need to remember these - they're automatic.

Terminal Mode Tip

For interactive commands like ,,i (Elixir REPL)

Exit terminal mode:

Ctrl + \ then Ctrl + n
Then use normal vim navigation to switch panes/buffers
Terminal mode captures all keys including Escape. Ctrl+\ Ctrl+n returns you to normal mode. From there you can switch windows, close the terminal, etc.

Markdown Preview

,mp

Toggle live preview in browser

Opens a live-updating preview as you type
markdown-preview.nvim opens a browser with live updates. Great for writing documentation and READMEs. Toggle off with ,mp when done.
