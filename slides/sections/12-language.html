Language-Specific Features

LocalLeader Commands

Press ,, for language actions

LanguageRunTestBuild
Go ,,r ,,t ,,b
Ruby ,,r ,,t -
JavaScript ,,r ,,t -
TypeScript ,,r ,,t -
Rust ,,r ,,t ,,b / ,,c
Each language uses the same keybindings but different commands. ,,r runs the current file, ,,t runs tests, ,,b builds. For Rust, ,,c runs cargo check which is faster than build.

Indentation Settings

Ruby2spaces

These are set automatically when you open a file
Indentation is configured per-language in ftplugin files. Go uses tabs as per its conventions. You don't need to remember these - they're automatic.

Markdown Preview

,mp

Toggle live preview in browser

Opens a live-updating preview as you type
markdown-preview.nvim opens a browser with live updates. Great for writing documentation and READMEs. Toggle off with ,mp when done.

  3. Use ,,t to run tests
  4. For Go/Rust: try ,,b to build
  5. Open a Markdown file
