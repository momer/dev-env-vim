gitsigns shows git status in the sign column. Hunks are chunks of changed code. ,hp shows a floating window with the diff for that hunk. ,hb shows who last modified the current line.

vim-fugitive

Git Commands

:Git / :G Open Git status
:Git blame Full file blame
:Git diff Diff view
:Gwrite Stage current file
:Gread Checkout current file
Fugitive is the definitive Git plugin for Vim. :G opens an interactive status view where you can stage with s. :Git blame opens a side-by-side blame view.

vim-rhubarb

GitHub Integration

:GBrowse

Opens current file on GitHub

Works with visual selection for line ranges!
Select lines, then :GBrowse to share a permalink.
:GBrowse is incredibly useful for sharing code with teammates. Select lines in visual mode and :GBrowse creates a permalink. Also works for PRs, issues, and commits.

Exercise 6: Git Workflow

  1. Make a small change to a file
  2. Use ]c to jump to the change
  3. Press ,hp to preview the hunk
  4. Press ,hb to see line blame
  5. Run :Git blame for full file blame
  6. Run :G to see git status
This exercise covers the most common git operations. Students should feel comfortable navigating changes and viewing history.
