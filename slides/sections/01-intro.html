Quick Start

git clone [repo] ~/.config/dev-env
cd ~/.config/dev-env/vim
make install

Prerequisites

  • neovim (0.9+)
  • node.js (for coc.nvim extensions)
  • ripgrep (for fast grep search)
  • Nerd Font (optional, for icons)
The Makefile handles symlinking configs and installing plugins. Node.js is required because coc.nvim extensions are npm packages. Ripgrep is used by fzf for blazing fast project searches.

Architecture

vim-plug Plugin manager
coc.nvim LSP client (completion, navigation, refactoring)
ALE Linting & formatting (eslint, prettier, gofmt)
Lua plugins Modern UI (lualine, nvim-tree, gitsigns)
This config uses a mix of VimScript and Lua plugins. coc.nvim handles LSP - this is where IDE features come from. ALE handles linting separately to avoid conflicts with coc. Lua plugins provide modern UI elements with better performance.
