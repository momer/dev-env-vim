nvim-tree

File Explorer

KeyAction
,E Toggle nvim-tree sidebar
,ef Find current file in tree
Use ,ef to quickly locate where you are in the project structure
nvim-tree provides a visual file explorer on the left side. It's useful for understanding project structure and file operations. ,ef is particularly useful when you've navigated deep into the codebase.

File Operations

aCreate file
dDelete
rRename
c / pCopy/Paste
IToggle hidden
These commands work inside the nvim-tree buffer. Press ? inside the tree for full help. Creating files with 'a' supports creating directories too (end with /).

Exercise 5: File Explorer

  1. Press ,E to open file tree
  2. Navigate with j/k
  3. Press a to create a test file
  4. Press d to delete it
  5. Press ,ef to find current file
  6. Press ,E to close
This exercise familiarizes students with basic tree operations. Emphasize that fzf is often faster for opening files. Tree is better for file operations and understanding structure.
