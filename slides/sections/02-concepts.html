All custom mappings start with comma

,, LocalLeader - language-specific commands
The leader key is the starting point for all custom mappings. We use comma because it's easy to reach and rarely used for editing. LocalLeader (double comma) is for language-specific commands like running tests.

Keybinding Discovery

which-key.nvim

  1. Press , (leader)
  2. Wait ~500ms
  3. Popup shows all available mappings
Pro Tip: Use this constantly when learning - no need to memorize everything!
which-key is your built-in cheat sheet. When you press leader and pause, it shows all available keybindings with descriptions. This is extremely helpful when learning the config.

Exercise 1: Discovery

  1. Open neovim
  2. Press , and wait
  3. Explore the popup menu
  4. Try ,f to find a file
  5. Try ,g to grep the codebase
This exercise gets students comfortable with leader key and which-key. After this, they should understand how to discover any keybinding on their own.
