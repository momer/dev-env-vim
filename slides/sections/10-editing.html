vim-commentary

Commenting Code

gcc Toggle comment on line
gc + motion Comment with motion (e.g. gcap)
gc (visual) Comment selection
Automatically uses correct comment syntax for each language
vim-commentary is incredibly useful for quick commenting. gc works like any vim operator - combine with motions. gcap comments the current paragraph (blank-line delimited). Works in visual mode for arbitrary selections.

vim-surround

Surround Text

cs"' Change surrounding  ("hi"→'hi')
ds" Delete surrounding
ysiw" Add surrounding to word
Works with: " ' ` ( ) [ ] { } < > t (tags)
In visual mode: S" surrounds selection
vim-surround is essential for editing quotes, brackets, tags. cs = change surrounding, ds = delete surrounding, ys = you surround. The 't' target works with HTML/XML tags. S in visual mode wraps the selection.

Other Utilities

Appendix

lexima.vim - Auto-closes brackets, quotes, pairs

vim-endwise - Auto-adds end in Ruby, Lua
These plugins work automatically in the background. lexima saves keystrokes when typing code. vim-repeat is essential for making . work properly. vim-endwise helps with Ruby and other languages with end keywords.

Exercise 7: Editing

  1. Type a word: hello
  2. Surround with quotes: ysiw"
  3. Change to backticks: cs"`
  4. Delete the backticks: ds`
  5. Comment a line: gcc
  6. Select lines visually, comment: gc
This exercise builds muscle memory for surround and comment operations. These become incredibly natural after practice.
