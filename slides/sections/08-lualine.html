Status Line

NORMAL main +2 ~1 init.vim
E:2 W:1 utf-8 unix vim 42% 12:5

Left Side

  • Mode indicator (colored)
  • Git branch
  • Diff stats (+added, ~changed, -removed)
  • Filename

Right Side

  • Diagnostics (errors/warnings)
  • Encoding & format
  • Filetype
  • Position (% and line:col)
lualine provides at-a-glance status information. The mode indicator changes color based on current mode. Git diff stats show how many lines are added/changed/removed. Diagnostics show error and warning counts from LSP.
